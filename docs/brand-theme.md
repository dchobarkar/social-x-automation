# brand-style.md

## Project Branding Guide

Minimal branding for the personal AI social automation project.
Goal: keep the UI clean, consistent, and easy to maintain.

---

## 1. Brand Personality

The project represents an **AI-powered social automation tool**.

Brand traits:

- Intelligent
- Efficient
- Developer‑friendly
- Minimal
- Productive

The interface should feel:

- clean
- lightweight
- focused on content
- not overly decorative

---
